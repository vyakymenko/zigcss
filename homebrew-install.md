# Homebrew Installation

## Using a Custom Tap

To install zcss via Homebrew, you can use a custom tap:

```bash
brew tap vyakymenko/zcss
brew install zcss
```

## Manual Installation

Alternatively, you can install directly from the formula:

```bash
brew install --build-from-source Formula/zcss.rb
```

## Updating

```bash
brew upgrade zcss
```
